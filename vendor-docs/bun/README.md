# Bun Documentation

Bun provides machine-readable documentation via the `llms.txt` standard.

- **Index (list of all pages):** https://bun.com/docs/llms.txt
- **Full docs (single file, ~1.9 MB):** https://bun.com/docs/llms-full.txt
- **API Reference:** https://bun.com/reference

AI agents can fetch `llms-full.txt` directly to get the complete documentation in one request.
