Fixed drag-and-drop of multiple images only attaching 1-2 files instead of all. The root cause was a race condition: parallel `resolveFilename` calls each read stale DOM state via `el.value`, and React 18 batched the resulting `setDescription` calls so only the last one survived. Switched to functional state updaters (`setValue(prev => ...)`) in both `CreateTaskModal` and `NoteItem` so each update correctly chains on the previous value.
