Migrated all CI to GitHub-hosted runners, removing the self-hosted macOS dependency. PR checks (lint, test, build-check) run in parallel on ubuntu-latest. Release builds now always produce all 3 platforms (macOS ARM64, macOS Intel, Linux) on GitHub-hosted runners with per-platform node_modules caching. Removed the `full-v*` tag and `full-release` script since every `v*` release is now a full cross-platform build.
