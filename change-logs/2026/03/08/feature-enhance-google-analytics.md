Enhanced Google Analytics 4 integration with full-featured tracking: added `first_open` event for new user detection, `user_agent` field for automatic OS/country/device resolution in GA4 standard reports, `app_update` event on version changes, `app_exception` for global error tracking, `screen_resolution` and `language` user properties, project lifecycle events (`project_added`, `project_removed`), settings change events (`theme_changed`, `locale_changed`, `settings_changed`), and session duration in heartbeat. Heartbeat interval reduced from 1 minute to 10 minutes to reduce event noise.
