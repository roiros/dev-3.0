Added "This is a PR review" toggle in the Create Task modal. When a branch is selected and review mode is enabled, the task description is pre-filled with a structured code review prompt. If the user already typed a description, the review prompt is prepended with a separator so nothing is lost. Turning off review mode restores the original text. Also fixed branch search to support queries containing "/" (e.g., "origin/dev"). Updated landing page and README with PR review mode feature and screenshot.
