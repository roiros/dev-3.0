Added split-view layout: clicking an active task card now opens the terminal alongside the Kanban board instead of navigating to a full-screen terminal. The Kanban shrinks to the left (~1.5 column widths) with a resizable divider, and the terminal occupies the right side. The divider width persists in localStorage. Double-click the divider to reset to default width. A new "full screen" button in the task info panel navigates to the full-screen terminal view. Press Escape to close the split and return to the full Kanban board.
