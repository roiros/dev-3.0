Added Cmd+Q and Cmd+, keyboard shortcuts. Cmd+Q shows a dialog informing the user that tmux sessions keep running in the background, with a "don't show again" checkbox (preference persisted in localStorage); once dismissed or skipped, it calls a new `quitApp` RPC that invokes `Utils.quit()`. Cmd+, opens global settings directly from any screen. Both shortcuts use a window capture-phase listener so they fire even when the ghostty-web terminal has focus.
