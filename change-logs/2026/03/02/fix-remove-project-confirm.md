Fixed "Remove project" button on Dashboard not working. The issue was that `window.confirm()` is not supported in Electrobun's WKWebView (it silently returns `false`). Replaced with the native `api.request.showConfirm()` RPC call, consistent with how `TaskCard` already handles confirmations. Also changed project removal from hard-delete to soft-delete (`deleted: true` flag) for future recoverability.
