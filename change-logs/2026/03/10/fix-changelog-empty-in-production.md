Fix changelog showing "No changelog entries yet" in production builds. The Electrobun 1.14 upgrade changed how app resources are bundled (tar archive instead of extracted files), breaking filesystem-based changelog.json reads. Added a build-time inlined TypeScript fallback: scripts/generate-changelog.ts now also writes src/bun/changelog-bundled.ts which the Bun bundler embeds directly into main.js, bypassing the need for runtime filesystem access.
