Add automatic Claude Code hooks (PreToolUse, PermissionRequest, Stop) to dev-3.0 worktrees for reliable task status transitions. Hooks are injected into `.claude/settings.local.json` per-worktree at launch time. PreToolUse sets "Agent is Working" on every tool call, PermissionRequest moves to "Waiting for You" when the agent needs permission approval, and Stop moves to "Review by User" when the agent finishes. A new `--if-status` conditional flag was added to the CLI `task move` command. The Claude SKILL.md was simplified to acknowledge hooks handle status automatically — generic agent skills (Cursor, Codex, etc.) retain the full manual instructions.
