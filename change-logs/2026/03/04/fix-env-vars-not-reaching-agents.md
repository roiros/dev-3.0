Fix environment variables not propagated to agent commands inside tmux sessions. All tasks share a single tmux server (`-L dev3`), so env vars passed to the tmux client process were lost for subsequent tasks. Now env vars are exported in wrapper bash scripts and propagated via `tmux set-environment` for split panes. Also added `CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1` as a default env var for all Claude-based agents.
