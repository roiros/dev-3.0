Redesigned coding agents to a two-level Agent + Configuration model. Each agent (Claude, Codex, Gemini, or user-created) now has multiple configurations with model selection, append prompt with Mustache-style template variables ({{TASK_TITLE}}, {{PROJECT_NAME}}, etc.), additional CLI arguments, environment variables, and base command override. Built-in agents are now editable but not deletable. Global Settings shows expandable agent cards with inline config editing. Project Settings has two dropdowns for agent and configuration selection.
